# 2014-games-checkers
A tool for testing thread performance of hardware using a game of life game.

## Compile
```bash
# compile java files
javac com/tashiv/goload/*.java

# package as runnable jar file
jar cfm goload.jar manifest/MANIFEST.MF com/tashiv/goload/*.class
```

## Usage
```bash
# run the jar file
java -jar goload.jar
```
